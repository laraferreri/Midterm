How to make your own LED mirror

One: Get a mirror

Find a mirror that you want to transform into a mirror with good lighting. You can choose any type of mirror with straight edges so that you can attach the lights.

Two: Order lights

Order strip lights off amazon that you want to use. Attach them using a sticky adhesive to the edges of the mirror. Control the lights using whichever device the light set comes with to have a well lit mirror.
